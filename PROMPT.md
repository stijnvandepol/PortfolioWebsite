# Opdracht: Bouw een premium portfolio pagina

Bouw een single-page portfolio website. Donker thema, premium uitstraling geïnspireerd op Apple/iOS designtaal. Puur HTML, CSS en JavaScript — geen frameworks. Mobile-first, volledig responsive. Taal: Nederlands.

De pagina moet aanvoelen als een gepolijst product: glaseffecten, gelaagdheid, zachte schaduwen, vloeiende overgangen, micro-interacties. Subtiele OS-hints (window controls, dock-navigatie, statusbar). Geen generiek template — iets dat je onthoudt.

## Layout

- **Sidebar** (links, sticky op desktop / inklapbaar op mobiel): profielfoto, naam, titel, contactinfo, social links
- **Main content** (rechts): wisselt tussen views via tabs, geen verticaal scrollen
- **Navigatie tabs:** Over Mij · Ontwikkeling · Portfolio · Blog · Contact
